# 压力择才：招聘模拟舱

一个基于 React、Vite 和 TypeScript 的 HR 抗压决策测评游戏。玩家需要在有限时间与 5 点查证资源下完成五名候选人的初评、证据核验、重评、沉没成本事件和最终录用。

## 启动

运行以下命令：

    npm install
    npm run dev

浏览器打开 Vite 输出的本地地址。快速测试模式为 3 分钟，正式模式为 15 分钟。

## 验证

    npm test -- --run
    npm run typecheck
    npm run build

## 快速完整流程

1. 选择“快速测试 · 3 分钟”。
2. 依次打开 A-E 五名候选人，为每人提交 T1。
3. 浅查 A，再深查 A，观察负面证据后的 HR 提醒。
4. 也可用 1 点浅查其他候选人；资源不足时深查按钮会禁用并说明原因。
5. 点击底部“测试：推进到最后 1 分钟”，触发沉没成本事件。
6. 作出沉没成本选择，进入最终决策并录用一人。
7. 查看报告并导出 JSON。

## 说明

- 核心规则位于 src/state 和 src/utils，UI 组件不承载评分算法。
- CSS 透视模拟舱仅用于入口氛围，不影响 2D 核心流程。
- RDI 为单人 MVP 的 0-100 加权指标，并保留原始数据以便未来接入群体 Z 值。
